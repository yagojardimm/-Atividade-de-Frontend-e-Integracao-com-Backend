# Backend APIs (Node.js & Flask)

Este diretório contém os códigos-fonte dos servidores de backend.

⚠️ **IMPORTANTE**: A execução manual com `npm start`, `npm run dev` ou `python app.py` **não é recomendada** como fluxo principal.

Toda a aplicação (frontend, backends e banco de dados) deve ser inicializada de forma integrada utilizando **Docker Compose**.

Consulte o arquivo **`README.md` na raiz do projeto** para obter as instruções completas de execução com:
```bash
docker-compose up --build
```

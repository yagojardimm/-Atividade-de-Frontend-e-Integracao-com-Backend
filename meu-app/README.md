# Frontend React

Este diretório contém o código-fonte do frontend desenvolvido com React e Tailwind CSS v4.

⚠️ **IMPORTANTE**: A execução manual com `npm start` ou `npm run dev` **não é necessária** para o fluxo principal de entrega.

Toda a aplicação (incluindo este frontend, os backends e o banco de dados) deve ser inicializada de forma integrada utilizando **Docker Compose**.

Consulte o arquivo **`README.md` na raiz do projeto** para obter as instruções de execução via Docker:
```bash
docker-compose up --build
```
